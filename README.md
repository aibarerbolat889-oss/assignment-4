## Description
Library Management System is a Java OOP project created for Assignment 4 (Milestone 2). The system manages books and library users, supports borrowing and returning books, and tracks availability.
The project demonstrates the use of:
Generics
Lambda expressions
Singleton, Builder, and Factory design patterns
The application is structured using clean object-oriented principles and can be easily extended.
